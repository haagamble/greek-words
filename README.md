# Greek Words App

A small React/Vite vocabulary quiz app for learning Modern Greek words.

The app lets learners choose a quiz by level or by broad topic, then gives a 10-question multiple-choice quiz. Greek nouns include their article, which helps learners see gender as part of the word.

## Features

- 20 vocabulary levels
- 20 broad topic groups
- Multiple-choice quizzes
- Greek noun articles shown with the word
- Distractors chosen from the same small category and level first, then falling back to broader matches when needed
- Vocabulary stored in plain JavaScript data files

## Project Structure

```text
src/
  main.jsx        Main React app
  styles.css      App styling

words.js          Main vocabulary list
big-topic.js      Broad topic definitions for topic quizzes
index.html        Vite entry HTML
vite.config.js    Vite configuration
```

## Vocabulary Data

Each word in `words.js` has fields like:

```js
{
  id: "water",
  type: "noun",
  level: 1,
  category: "food-drink",
  greek: "νερό",
  article: "το",
  gender: "neuter",
  english: "water"
}
```

The app uses:

- `level` for level-based quizzes
- `category` for narrow grouping and good multiple-choice distractors
- `big-topic.js` for the 20 broad user-facing topic choices

## Getting Started

Install dependencies:

```bash
npm install
```

Start the development server:

```bash
npm run dev
```

Build for production:

```bash
npm run build
```

Preview a production build:

```bash
npm run preview
```

## Notes

The source vocabulary and topic files are intended to be edited directly. After changing the data, run `npm run build` to make sure the app still compiles.

The `dist/` folder is generated by the build and does not need to be committed. The `node_modules/` folder is generated by `npm install` and should not be committed.
